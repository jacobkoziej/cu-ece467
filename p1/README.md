# p1

> Text Categorization: Rocchio/TF*IDF


## Usage

Help:

```
./p1.py -h
```

Training:

```
./p1.py train -i input_labels -d output_database
```

Testing:

```
./p1.py test -d trained_database -i input_list -o output_labels
```

Test Generation:

```
./p1.py testgen -i labeled_corpus -o output_prefix
```
